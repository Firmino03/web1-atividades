## Web 1 - 2º Período IPI ##

Este repositório reúne as atividades práticas desenvolvidas na disciplina Desenvolvimento Web I, do 2º período do curso de Informática para Internet (IPI).
Os exercícios e projetos aqui publicados são implementados principalmente em PHP, abordando conceitos essenciais para criação de aplicações web dinâmicas.
